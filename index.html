Health Work Flow Management System

A full-stack healthcare management platform enabling CRUD operations for patient records, appointments, prescriptions, feedback, and medical documents.

Key Features:
  • Prescription Management: Doctors can create, view, and manage prescriptions with real-time drug name suggestions from an integrated drug database.
  • AI Drug Safety System: AI checks patient medical history and provides real-time warnings to prevent risky drug interactions during prescription entry.
  • Document Management: Patients can securely upload medical documents to the doctor.
  • Online Appointment Booking: Patients can book consultations with doctors, selecting time slots and receiving auto-generated meeting within the system.
  • AI-Powered Feedback Reports: Patients submit feedback post-consultation, and the system generates AI-driven reports to help doctors analyze trends.
  • Doctor–Patient Chat System: Encrypted in-app messaging allows secure post-visit communication.
  • Role-Based Access Control: System access is strictly separated by user roles.
  • Cloud Storage: All sensitive documents and records are securely stored using scalable cloud platforms.
My Contribution:
  • Developed the Prescription Management System
  • Implemented the AI-based drug warning module using patient history
  • Built the drug suggestion system integrated with an external drug database
